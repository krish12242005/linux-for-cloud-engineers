# Linux Basic Commands

## Check Current Directory
pwd

## List Files
ls

## Change Directory
cd foldername

## Create Folder
mkdir test

## Create File
touch file.txt

## Delete File
rm file.txt

## Copy File
cp file1.txt file2.txt

## Move File
mv file1.txt folder/

## View File Content
cat file.txt

## Check Disk Usage
df -h

## Check RAM Usage
free -m
